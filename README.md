# bookings
Аналитическое исследование данных о бронированиях за 2025 год на основе датасета с информацией о клиентах, субпродуктах (avia, hotel, taxi, train) и транзакциях. Цель — выявить ключевые бизнес-метрики: динамику оборота (GMV), поведение клиентов по тирам, отмены бронирований и retention новых пользователей.
